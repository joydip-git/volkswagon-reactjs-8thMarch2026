TypeScript:
---------------
superset of JavaScript
development mode language
static type checking
strict typing


TS -> convert -> JS [transpiler -> TSC]

Note: extra features of TS (such as, interface, type, generic, extra data types [any, unknown etc.]) ae NOT converted to JS during transpilation

install typescript:
-----------------------------
>npm i typescript -D

create tsconfig.json file:
-----------------------------
>.\node_modules\.bin\tsc --init

"use strict"

Reat with TypeScript:
----------------------------
StrictMode: 
a built-in component. 
used during development time and wraps the whole react app (root component App)
Lets you find common bugs in your components early during development.

Header({messageValue:message})

	messageValue:string
}
