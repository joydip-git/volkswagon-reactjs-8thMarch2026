React JS:
---------------
a JS library to create client-side UI based web app.
It is used to create Single Page Application (SPA) type app
a React UI is composed of several components
each component is an atomic unit which contains
	- HTML design (dynamic)
	- data and event handlers (code)
	- styles

It offers "binding" feature to render data from code in the template (UI/Design/View) as well as vice versa

	create-react-app (deprecated)
	vite (pronounced as veet)

mounting:
 the very first time the whole VDOM converted to HTML DOM and appended in the xisting DOM tree for the HTML file

unmounting: 
taking off any html (react element) from the DOM

updating: 
due to change in data in a component React JS creates a new VDOM version and compares the same with the previous version to find out which component data has been updated, in order to update the corresponding HTML element in the actual HTML DOM displayed in the browser
