React JS:
---------------
a JS library to create client-side UI based web app.
It is used to create Single Page Application (SPA) type app
a React UI is composed of several components
each component is an atomic unit which contains
	- HTML design (dynamic)
	- data and event handlers (code)
	- styles

It offers "binding" feature to render data from code in the template (UI/Design/View) as well as vice versa

CLI tool:
	create-react-app (deprecated)
	vite (pronounced as veet)
	Parcel

framework tools:
	Next JS
	Gatsby
	React Router
	Expo
	TanStack
	RedwoodSDK

Vite to create React app:
	>npm i vite --localtion=global
	>vite ...

without installing vite:
	>npm create vite@latest

React packages:
	react
	react-dom


modules in JS:
------------------
multiple script tag

creating modules in different way: module format

1. Common JS format: (default module format for Node JS app)
a. exporting members: module.exports= { prop1:member1, prop2:member2,..}
b. importing members: const moduleexports = require('module-path')

2. Asynchronous module definition format (AMD)

require('',[],function(){
	
})

3. Universal module definition format (UMD)

4. System JS

5. ES6 module format (ESM)
a. exporting members: export function add(a,b){}
b. importing members: import { add:add } from './utility'
